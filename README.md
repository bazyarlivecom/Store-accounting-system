# 🚀 سیستم حسابداری و مدیریت مالی یکپارچه (Enterprise ERP & Accounting System)

یک نرم‌افزار جامع، پیشرفته و یکپارچه سازمانی (ERP) برای مدیریت کامل فرآیندهای حسابداری، خزانه‌داری، زنجیره تأمین، انبارداری، منابع انسانی، و گزارش‌گیری‌های مالی هوشمند. این سامانه با ظاهری بسیار مدرن و کارآمد توسعه یافته و اکنون به **پوسته گوگل جیمیل (Gmail Theme)** با کارایی بهینه مجهز شده است.

---

## 🎨 پوسته‌ها و تجربه کاربری (Themes & UX)

این سامانه از یک سیستم مدیریت تم فوق‌العاده پیشرفته و قابل سوئیچ بهره می‌برد که کاربر می‌تواند در بخش **تنظیمات سیستم** تم دلخواه خود را فعال کند:

1. **پوسته کلاسیک سرمه‌ای (Classic Indigo Theme)**: ظاهر پیش‌فرض سازمانی، بسیار پایدار، شیک و مناسب برای مانیتورهای عریض با کنتراست رنگی متعادل.
2. **پوسته گوگل جیمیل سرخ (Google Gmail Theme)**: الهام گرفته از نسخه جدید وب‌سایت Gmail گوگل با مشخصات زیر:
   - **پالت رنگی خاکستری سرد و سرخ جیمیل** (`bg-[#f6f8fc]` و رنگ دکمه‌ها با رنگ شناسه `#b3261e`).
   - **دکمه شناور "ایجاد جدید" (Compose Button)**: بازکننده مودال ابزار میانبر سریع برای ایجاد فاکتورها، اسناد دریافت/پرداخت و انتقال وجه با یک کلیک در هر کجای برنامه.
   - **نوار جستجوی پیشرفته بالای صفحه (Google Search Bar)**: نوار مجهز و یکپارچه برای جستجوی پویا در فاکتورها، اشخاص و تراکنش‌ها.
   - **سایدبار منحنی و منعطف**: منوهایی با گوشه‌های گرد جیمیلی و افکت‌های هاور مدرن.

## ✨ ماژول‌ها و امکانات سیستم (Core Modules)

این نرم‌افزار مجموعه‌ای از ماژول‌های کاملاً یکپارچه است که جریان داده را به طور خودکار میان بخش‌های مختلف هدایت می‌کند:

### ۱. فاکتورها و خرید و فروش (Sales & Procurement)
* **صدور فاکتور فروش (Invoice)** و **فاکتور خرید (Purchase)** با جزئیات کامل محاسباتی.
* **برگشت از فروش** و **برگشت از خرید** برای مرجوعی کالاها.
* محاسبه خودکار تخفیف ردیفی و کلی، مالیات بر ارزش افزوده، و مبالغ نهایی.
* چاپ پیشرفته با فرمت‌های مختلف (مانند کاغذ A4 و A5 و فیش‌پرینتر حرارتی) به همراه امضاها و لوگوی اختصاصی فروشگاه.

### ۲. خزانه‌داری و مدیریت چک (Treasury & Cheques)
* **ثبت اسناد دریافت وجه (قبض رسید)** و **پرداخت وجه (برگه پرداخت)** به صورت نقد یا چک.
* **دفتر چک‌های دریافتی و پرداختی** با وضعیت‌های مختلف (در جریان وصول، وصول شده، برگشتی، خرج شده).
* **تقویم و نمودارهای چک**: پایش سررسید چک‌ها روی تقویم شمسی پویا و نمودارهای آماری دقیق.
* امکان انتقال وجه بین صندوق‌ها و بانک‌ها (سند جابجایی).

### ۳. حسابداری دوطرفه و اسناد (Double-Entry Bookkeeping)
* **ثبت و صدور اسناد حسابداری دستی و خودکار** (اسناد خودکار همزمان با ثبت فاکتور و رسیدها تولید می‌شوند).
* **سرفصل‌های حسابداری (Chart of Accounts)**: ساختار درختی و ۳ سطحی (گروه، کل، معین) منطبق با استانداردهای مالی.
* **سال‌های مالی چندگانه (Financial Years)**: ایجاد سال‌های مالی جدید، بستن حساب‌ها و انتقال مانده به سال جدید به صورت خودکار.
* **تراز آزمایشی و سود و زیان**: دریافت سود و زیان ناویژه، سود عملیاتی و گزارش‌های جامع ترازنامه.

### ۴. انبارداری و زنجیره تأمین (Inventory Management)
* **تعریف انبارها** به تعداد نامحدود.
* **ثبت رسید انبار (ورود کالا)** و **حواله انبار (خروج کالا)**.
* کنترل خودکار موجودی کالاها با امکان غیرفعال‌سازی موجودی منفی (کنترل موجودی سخت‌گیرانه).
* **انبارگردانی (Stocktaking)**: ثبت مغایرت‌های انبار و اصلاح هوشمند کاردکس کالاها.

### ۵. مدیریت طرف حساب‌ها (Contact Management)
* دسته‌بندی اشخاص به گروه‌های دلخواه (مشتریان، همکاران، تامین‌کنندگان، واسطه‌ها).
* **سیستم نام مستعار (Alias)**: نمایش نام‌های مستعار و کدهای شناسایی طرف حساب به صورت بولد و برجسته در لیست‌ها برای تسریع در شناسایی هویت تجاری اشخاص.
* دفتر معین اشخاص (صورتحساب تفصیلی) با قابلیت فیلتر بر اساس تاریخ و نوع تراکنش‌ها.

### ۶. حقوق و دستمزد (HR & Payroll)
* تعریف پرسنل و کارمندان به همراه تعیین نقش‌ها و سمت‌های سازمانی.
* محاسبه کارکرد، اضافه کار، کسری کار، حق مسکن، بن کارگری و بیمه.
* صدور خودکار **فیش‌های حقوقی (Payslips)** و ثبت هزینه‌ها در سرفصل‌های معین حسابداری.

## 🛠️ معماری فنی و دیتابیس (Technical Stack)

* **فرانت‌اند**: 
  - **React 18 & Vite** با تایپ‌اسکریپت (TypeScript) جهت امنیت بالای کدنویسی.
  - **Tailwind CSS** برای پیاده‌سازی استایل‌های کاستومایز و بسیار سریع.
  - **Motion (Framer Motion)** برای انیمیشن‌های روان انتقالی و باز شدن مودال‌ها.
  - **Recharts** جهت تصویرسازی و ترسیم نمودارهای تحلیلی مالی.
* **بک‌اند**:
  - **Express (Node.js)** به عنوان لایه وب‌سرویس و API.
  - **Node:SQLite (DatabaseSync)**: بهره‌گیری از ماژول بومی، فوق‌العاده سریع و همگامِ دیتابیس SQLite در نسخه‌های جدید Node.js که بدون نیاز به درایورهای سنگین خارجی کار می‌کند.
  - **Bcrypt & JWT Auth**: فرآیند احراز هویت بسیار امن با کلمه‌های عبور رمزنگاری شده و توکن‌های دوگانه دسترسی (Access/Refresh Tokens).
  - **سامانه مدیریت پایگاه‌داده اختصاصی**: امکان ایجاد بکاپ لحظه‌ای از فایل `database.sqlite` و بازیابی آن با یک کلیک در پنل ادمین.

## 🏃‍♂️ راهنمای راه‌اندازی سریع پروژه (Getting Started)

### ۱. نصب وابستگی‌ها
مطمئن شوید که نسخه‌ای از Node.js (ترجیحاً نسخه ۱۸ به بالا) روی سیستم شما نصب است. سپس در پوشه پروژه دستور زیر را بزنید:
```bash
npm install
```

### ۲. اجرای نسخه توسعه (Development)
برای اجرای سرور Express و فرانت‌اند Vite به صورت همزمان:
```bash
npm run dev
```
سپس مرورگر خود را باز کرده و آدرس `http://localhost:3000` را وارد کنید.

### ۳. ساخت و اجرای نسخه نهایی (Production)
برای بیلد کردن فرانت‌اند به کدهای بهینه جاوااسکریپت و باندل کردن سرور با esbuild:
```bash
npm run build
```
سپس پروژه را در محیط واقعی استارت کنید:
```bash
npm start
```

## 🗂️ ساختار فایل‌های اصلی پروژه
* `/server.ts` - سرور اصلی Express، مدیریت توکن‌های JWT، کوکی‌ها، آپلود فایل، ذخیره‌سازی داده‌ها در SQLite و تعریف APIها.
* `/src/App.tsx` - هسته مرکزی رابط کاربری SPA شامل تمامی تب‌ها، مودال‌ها، محاسبات فاکتور و لایوت‌های داینامیک پوسته‌ها.
* `/src/types.ts` - تعاریف یکپارچه تایپ‌اسکریپت برای آبجکت‌های مالی، فاکتورها، کاربران، اسناد و انبارها.
* `/src/services/dataService.ts` - لایه کلاینت برای تبادل داده با سرور و همگام‌سازی اتوماتیک تنظیمات برنامه.

توسعه یافته با استانداردهای نوین طراحی کاربری جهت سهولت استفاده در کسب‌وکارهای فروشگاهی و شرکتی. 🚀
