# سیستم حسابداری یکپارچه

یک نرم‌افزار حسابداری جامع، ساده و قدرتمند که به صورت اختصاصی برای مدیریت اشخاص، کالاها، حساب‌های بانکی، صندوق‌ها، فاکتورهای خرید و فروش، اسناد دریافتی و پرداختی و همچنین فیش‌های حقوقی توسعه داده شده است.

## ✨ امکانات کلیدی

- **مدیریت طرف حساب‌ها**: ثبت و ویرایش اطلاعات مشتریان، تامین‌کنندگان و کارمندان.
- **مدیریت کالا و خدمات**: تعریف محصولات با قیمت‌های پایه و نگهداری سوابق.
- **مدیریت مالی و خزانه‌داری**: 
  - ثبت حساب‌های بانکی و صندوق‌ها.
  - ثبت رسیدهای دریافت و پرداخت با امکان انتخاب مبداء/مقصد (بانک، صندوق).
  - مدیریت و صدور فیش‌های حقوقی برای کارمندان.
- **خرید و فروش (فاکتورها)**:
  - صدور فاکتورهای خرید و فروش (با قابلیت تولید شماره فاکتور خودکار).
  - چاپ فاکتورها با فرمت استاندارد.
  - محاسبه خودکار تخفیفات، مالیات و مبالغ کل.
- **گزارشات و ابزارها**:
  - مشاهده دفتر کل و صورتحساب اشخاص.
  - گزارشات جامع فعالیت‌های مالی.
  - ابزارهای داخلی نظیر ماشین حساب و چک‌لیست کاربردی.
- **مدیریت پایگاه داده**:
  - تمامی داده‌ها به صورت امن در سرور در قالب فایل `database.json` ذخیره می‌شوند.
  - امکان مشاهده حجم داده‌ها، تهیه نسخه پشتیبان (Backup) و بازیابی (Restore) اطلاعات با یک کلیک محیط کاربری.

## 🚀 تکنولوژی‌های استفاده شده

- **محیط کاربری (Front-end)**: React 18, Vite, Tailwind CSS 
- **انیمیشن‌ها و گرافیک**: Motion for React, Lucide React
- **تاریخ شمسی**: `react-multi-date-picker`
- **ساختار بک‌اند (Back-end)**: Node.js (Express) با طراحی فایل‌محور جهت حذف وابستگی به پایگاه‌داده‌های سنگین و پیچیده.

## 🛠️ راهنمای اجرا و نصب پروژه

### پیش‌نیازها
- Node.js (نسخه ۱۸ یا بالاتر)

### مراحل نصب و اجرا

۱. ابتدا وارد مسیر پوشه پروژه شوید.
۲. پکیج‌های پیش‌نیاز را از طریق npm نصب کنید:
```bash
npm install
```
۳. برای اجرای نسخه توسعه (به همراه سرور و محیط ابزار کار):
```bash
npm run dev
```

برنامه روی آدرس زیر در مرورگر قابل دسترسی خواهد بود:
`http://localhost:3000`

### ساخت نسخه نهایی (Production Build)
برای ایجاد نسخه بهینه‌شده جهت اجرا در سرور نهایی:
```bash
npm run build
```
سپس برای راه‌اندازی سرور حالت نهایی:
```bash
npm start
```

## 🗄️ ساختار داده‌ها (پایگاه داده محلی)

داده‌های برنامه (فاکتورها، مشتریان و...) به صورت ساختاریافته در یک فایل متنی امن به نام `database.json` در مسیر اصلی پروژه ذخیره می‌شوند. با مراجعه به تب **مدیریت دیتابیس** در سایدبار پنل نرم‌افزار، می‌توانید همواره وضعیت سلامت این فایل، تعداد رکوردها و حجم آن را کنترل کنید و در پایان روز از اطلاعات خود خروجی (Export) بگیرید.
